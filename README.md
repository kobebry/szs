회원가업 -> 로그인(토큰 취득) -> Swagger Authorize -> Bearer [토큰] 입력 후 Authorize 클릭

이후 유저정보 조회/스크랩/결정세액, 퇴직연금세액공제금액 계산 Api 테스트 해보 실 수 있습니다.
